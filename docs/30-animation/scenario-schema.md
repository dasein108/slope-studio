# Scenario Schema — `01_script.json`

Authoritative reference for authoring a scenario (by hand or LLM). The pipeline
(`studio script` or a hand-authored file) must produce exactly this shape. Models:
`studio/models.py` (`Script`, `Scene`).

## Top-level object

```jsonc
{
  "topic": "string",            // short subject line
  "duration_s": 150,            // TARGET; sets scene COUNT (≈duration/6). Final length follows narration (±a short tail)
  "aspect": "9:16",             // 9:16 vertical (default) | 16:9 landscape/classic YouTube | 1:1 | 4:5 | 4:3 | 21:9 — every stage adapts (studio/canvas.py)
  "voice": true,                // generate voiceover + captions
  "voice_name": "woman",        // narrator voice: man|woman|cartoon|narrator (voices.md)
  "tone": "neutral",            // neutral|serious|mystical|friendly|sad|excited
  "style": "string",            // tone/style note (informs prompts)
  "character": "string",        // ONE reusable subject/style description; reuse verbatim in every visual_prompt for consistency
  "music": "",                  // background-music MOOD phrase for whole video (instrumental); "" = none. Generated by the `audio` stage, ducked under narration. providers/audio.py
  "scenes": [ Scene, ... ],
  "title": "string #Shorts",    // for publishing
  "description": "string",
  "hashtags": ["#shorts", "..."]
}
```

## Scene object

```jsonc
{
  "id": 1,                      // 1-based, unique, sequential
  "start_s": 0,                 // tiles [0, duration_s] with no gaps/overlaps
  "end_s": 10,                  // end_s of last scene == duration_s
  "visual_prompt": "string",    // image prompt; PREPEND the `character` string for consistency
  "narration": "string",        // ~2.6 words/sec of speech for this scene (drives clip length)
  "on_screen_text": "SHORT",    // big headline (kinetic/card); keep <= 3 words
  "motion_hint": "slow push-in",// free-text; used as i2v prompt + hints preset choice
  "priority": 0,                // 0=auto; higher = animate with AI budget first (auto strategy)
  "image_role": "",             // "hero" = character/main person → quality model (+char ref); "bg" = background/overlay → cheap model (~4x cheaper); "" = stage default. Controls stage-2 cost.

  // --- presentation (all optional; see docs/30-animation/) ---
  "animator": "kenburns",       // kenburns | motion-driftright|driftleft|driftup|driftdown|zoomin|zoomout|pulse | kinetic | parallax | blurred-parallax | slice | static | puppet | talkinghead | manim
  "atmosphere": "",             // DEFAULT "". weather/particle overlay: rain|snow|embers|blood|petals|leaves|wind|fog. Set ONLY when the scene LITERALLY depicts it (real snow/rain/fire/fog outdoors). NEVER on indoor/portrait/studio/diagram/neutral scenes or for "mood". Snow in an office is a bug. (atmosphere.md)
  "fx": [],                     // look post-passes (after atmosphere): grain|vignette|chroma|glitch|sunrise|sunset|godrays|oldfilm|flash[-white|-yellow|-red|-black]. Pick ONE coherent look for the whole video; don't sprinkle different fx per scene. godrays needs a real light source; chroma/glitch never on calm/realistic pieces. (effects/ffmpeg-recipes.md)
  "transition": "cut",          // into this scene: cut|fade|fadeblack|dissolve|wipeleft|wiperight|wipeup|wipedown|slideleft|slideright|slideup|slidedown|smoothleft|smoothright|circleopen|circleclose|radial|pixelize|zoomin
  "transition_dur": 0.4,        // seconds (0 -> default 0.4)
  "manim_code": "",             // only for animator:"manim" — body of construct() (see manim.md)
  "mouth_set": "",              // only for animator:"talkinghead" — sprite set under assets/mouths/<set>/ ("" -> drawn default); effects/talking-head.md
  "mouth_xy": [],               // only for animator:"talkinghead" — [x_frac, y_frac] mouth anchor on the face (0-1; "" -> [0.5, 0.6])
  "limbs": [                    // only for animator:"puppet" — per-limb joint rotation (hand up/wave); effects/puppet.md
    {"box":[0.35,0.32,0.42,0.6], "pivot":[0.39,0.34], "move":"raise", "amp":120, "period":0.8, "phase":0}
  ],
  "tone": "",                   // optional per-scene voice tone override (else Script.tone); voices.md

  // --- sound effects (optional; generated by the `audio` stage) ---
  "sfx": [                      // 0-2 one-shot effects; empty for most scenes
    {
      "prompt": "metallic sword clash, sharp",  // vivid sound description
      "at": 0.5,                // seconds INTO this scene to trigger
      "dur": 0.8,               // requested length; accents 0.2-2s, hard max 5s
      "gain_db": -10            // level vs narration mix (-15 quiet .. -6 prominent)
    }
  ]
}
```

## Authoring rules (enforced / strongly recommended)

1. **Timing tiles the timeline.** `scenes[0].start_s == 0`; each `start_s ==` previous
   `end_s`; last `end_s == duration_s`. `studio` validates and warns
   (`Script.validate_timing`). Final video length follows *narration*, not these
   numbers — but keep them coherent (~6–12s/scene).
2. **Narration drives real length.** ~2.6 words/sec. A 10s scene ≈ 24–28 words. The
   `narrate` stage measures actual TTS length and sets each clip to match.
3. **Consistency:** put the recurring subject/style in `character` and copy it verbatim
   at the start of every `visual_prompt`.
4. **`on_screen_text`** short and punchy — it's the kinetic/card headline.
5. **Pick `animator` per scene by content** — `manim` ONLY for educational/scientific
   diagrams (NEVER art/story — it reads as schematic); subject+landscape → `parallax`;
   dramatic reveal/entrance/beheading → `slice`; calm beat → `static`;
   text-free illustration → `kinetic`; directional → `motion-drift*`. See
   [`README.md`](README.md#animator-decision-matrix). **Only these names are wired** — an
   unknown `animator` falls back to `kenburns`. For richer effects (rain, fire, fog,
   sunrise, morphs, kinetic typography…) use `animator:"manim"` + `manim_code`, or see the
   [`effects/`](effects/README.md) index for the full catalog + recipes.
6. **Pick `transition` by narrative flow** — see [`transitions.md`](transitions.md).
7. **First scene's `transition` is ignored** (nothing precedes it).
8. **Audio is opt-in by content.** Only add scene `sfx` for a clear diegetic moment
   (clash, whoosh, slam, breath) or one sharp story accent — most scenes have none.
   Accent SFX should be 0.2-2s and must never exceed 5s; long bells, drones, rumbles,
   and ambiences belong in top-level `music` or deliberate environment, not random
   punctuation. Set top-level `music` to one instrumental mood phrase, or `""` for silence.
   Cost only accrues when these are present and a paid provider is selected
   (balanced/premium tiers); the `stub` script provider leaves both empty so default runs
   stay $0.

## Minimal valid example

```json
{
  "topic": "why ice floats", "duration_s": 20, "aspect": "9:16", "voice": true,
  "style": "fun explainer", "character": "clean flat science illustration, blue palette, 9:16",
  "scenes": [
    {"id":1,"start_s":0,"end_s":10,"visual_prompt":"clean flat science illustration, blue palette, 9:16, an ice cube floating in a glass","narration":"Ice floats, and that tiny fact is why life under frozen lakes survives winter.","on_screen_text":"ICE FLOATS","animator":"kenburns","transition":"cut"},
    {"id":2,"start_s":10,"end_s":20,"visual_prompt":"clean flat science illustration, blue palette, 9:16, water molecules forming an open hexagonal lattice","narration":"When water freezes, its molecules lock into an open lattice that is less dense than liquid water.","on_screen_text":"OPEN LATTICE","animator":"manim","transition":"circleopen","transition_dur":0.6,"manim_code":"hexagon = RegularPolygon(n=6, color=BLUE).scale(2)\n        self.play(Create(hexagon))\n        self.play(hexagon.animate.scale(1.3))\n        self.wait()"}
  ],
  "title": "Why Ice Floats #Shorts", "description": "...", "hashtags": ["#shorts","#science"]
}
```

## How fields flow through stages

| field | consumed by |
|-------|-------------|
| `visual_prompt`, `character`, `aspect`, `image_role` | stage 2 visuals (hero→quality model+ref, bg→cheap model) |
| `narration`, `voice`, `voice_name`, `tone`, scene `tone` | stage narrate (per-scene TTS → clip durations + captions) |
| `animator`, `motion_hint`, `manim_code`, `atmosphere`, `priority` | stage 3 clips |
| `transition`, `transition_dur` | stage 4 stitch |
| `on_screen_text` | `card` image + `kinetic` headline |
| scene `sfx`, top-level `music` | stage `audio` (generate) → stage `voice` (overlay + sidechain-duck) |
| `title`, `description`, `hashtags` | stage 7 publish |

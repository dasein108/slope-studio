# Recipes

Opinionated, end-to-end presets for a specific *kind* of video — the tuned combination of
provider, animator, tone, transition, and pacing knobs that the generic pipeline docs describe
individually. Each recipe is a "if you want X, do exactly this" playbook.

| Recipe | For |
|--------|-----|
| [poetry.md](poetry.md) | Spoken-verse / poetry Shorts — slow delivery, drifting imagery, soft cross-fades. |

Add a recipe when a format needs a repeatable, non-obvious combination of settings worth
capturing for next time.

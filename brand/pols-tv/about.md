## How to apply (YouTube Studio — ~3 min)
1. studio.youtube.com → confirm you're on the **right channel** (top-right avatar → Switch account).
2. **Customization → Basic info:** set channel **name** to `Pols TV`; paste the **Description**.
3. **Customization → Branding:** upload **Picture** = `profile_800.png`; **Banner** = `banner_2560x1440.png`.
4. **Settings → Channel → Keywords:** paste the keywords. → **Publish**.

Note: profile photo can't be set via API (Google-account avatar) — upload in Studio.
Banner + description can be automated via `channels.update` if wanted.

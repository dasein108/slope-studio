## How to apply (YouTube Studio — manual, ~3 min)
1. studio.youtube.com → make sure you're on the **right channel** (top-right avatar → Switch account).
2. **Customization → Branding:** upload **Picture** = `profile_800.png`; **Banner image** = `banner_2560x1440.png`.
3. **Customization → Basic info:** paste the **Description** above; add the tagline as the first line.
4. **Settings → Channel → Keywords:** paste the keywords.
5. Click **Publish** (top-right).

Note: the profile photo CANNOT be set via the API (Google account avatar) — upload it in
Studio. Banner + description *can* be automated later via `channels.update` if you want.
